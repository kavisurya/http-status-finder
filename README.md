### Usage
```
-f -> input file example : /home/rao/urls.txt 
-d -> delay for each requests 
-p -> Your project name


python3 httpstatusless.py -f /home/rao/urls.txt -d 0 -p 'projectname'

```

### Time based checking

```
python3 httpstatusless.py -f /home/rao/urls.txt -d 5 -p 'projectname'

```
